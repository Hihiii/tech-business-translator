# Domain KPI Mapping

Use this guide to translate technical metrics into domain-specific business impact.

## SaaS

| Technical Signal | Business KPI |
|---|---|
| Login failure | activation, retention, account access |
| Slow dashboard | user engagement, renewal risk |
| API downtime | enterprise SLA, partner trust |
| Release delay | roadmap confidence, expansion revenue |
| Technical debt | feature velocity, maintenance cost |

## E-commerce

| Technical Signal | Business KPI |
|---|---|
| Checkout errors | GMV, conversion rate, abandoned carts |
| Payment latency | purchase completion, customer frustration |
| Inventory sync issue | oversell risk, fulfillment cost |
| Search outage | product discovery, conversion |
| CDN issue | page speed, bounce rate |

## Fintech

| Technical Signal | Business KPI |
|---|---|
| Transaction failure | payment completion, customer trust |
| Ledger inconsistency | reconciliation risk, audit exposure |
| Authentication issue | account access, fraud controls |
| Security vulnerability | regulatory exposure, breach risk |
| Delayed reporting | compliance deadlines, operational risk |

## Healthcare

| Technical Signal | Business KPI |
|---|---|
| Availability issue | care continuity, service reliability |
| Privacy control issue | patient data risk, compliance exposure |
| Audit log gap | traceability, regulatory readiness |
| Integration failure | clinical workflow disruption |
| Data delay | operational decision quality |

## B2B Enterprise

| Technical Signal | Business KPI |
|---|---|
| SLA breach | renewal risk, credits, escalation |
| API instability | customer workflow disruption |
| Support ticket spike | account health, support cost |
| Migration delay | onboarding timeline, revenue recognition |
| Security finding | procurement blocker, trust risk |

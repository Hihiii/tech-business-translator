# Example: Bilingual Progress Report

## Prompt

Write a bilingual progress report for PM and leadership:

> Search relevance project: shipped typo tolerance and synonym support. Ranking model still needs evaluation. Launch date may slip one week because data labeling is delayed. Current conversion uplift in experiment is 3.2%.

## Expected Traits

- Audience: PM / leadership.
- Template: progress report.
- Preserve 3.2% uplift and one-week risk.
- Explain data labeling as a dependency, not as a technical excuse.

## English Sample

The search relevance project is delivering early business value, with the current experiment showing a 3.2% conversion uplift. Typo tolerance and synonym support have shipped. The main schedule risk is a data-labeling delay that may push launch by one week while the ranking model is evaluated.

## Traditional Chinese Sample

搜尋相關性專案已開始展現商業價值，目前實驗顯示轉換率提升 3.2%。拼字容錯與同義詞支援已完成上線。主要時程風險是資料標註延遲，可能使正式發布延後一週，因為排序模型仍需完成評估。

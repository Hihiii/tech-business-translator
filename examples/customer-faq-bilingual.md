# Example: Bilingual Customer FAQ

## Prompt

Create a bilingual customer FAQ:

> A third-party analytics SDK collected event metadata for 2 hours longer than configured. No payment data. User IDs included. Data deletion requested from vendor.

## Expected Traits

- Legal-safe wording.
- Preserve 2-hour window, no payment data, user IDs, vendor deletion request.
- Avoid saying "breach" unless confirmed.

## English Sample

### What happened?

A third-party analytics tool collected event metadata for two hours longer than our configured window.

### Was payment data involved?

Current evidence indicates that payment data was not involved. The collected metadata did include user IDs.

### What are you doing?

We have requested deletion of the affected data from the vendor and are reviewing the configuration controls that allowed the extended collection window.

## Traditional Chinese Sample

### 發生了什麼？

第三方分析工具比我們設定的時間多收集了兩小時的事件中繼資料。

### 是否涉及付款資料？

依目前證據判斷，未涉及付款資料。被收集的中繼資料包含用戶 ID。

### 你們正在做什麼？

我們已要求供應商刪除受影響資料，並正在檢視導致收集時間延長的設定控管。
